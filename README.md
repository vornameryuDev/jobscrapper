# jobscrapper
jobscrapper

# 개요
1. 여러 취업 포털 들어가서 찾는것이 귀찮음
2. 한번에 보고 싶어서 scrapper 개발
3. 대상포털: SARAMIN, JOBKOREA
4. flask활용 웹개발
   1) 검색 -> 대상포털에 키워드 전달
   2) 스크래핑 진행(회사명, 제목, 경력, 학력, 링크)
   3) 한번 검색 진행되면 해당 내용 저장 -> 속도개선
   4) 엑셀 다운로드기능 추가
